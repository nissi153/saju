# 사주팔자 계산기 (음력 지원)

생년월일시를 입력하면 사주팔자, 오행 분석, 십신, 대운, 올해 운세를 알려주는 프로그램입니다.

## 설치

```
pip install korean-lunar-calendar
```

## 사용법

### 대화형 모드

```
python saju.py
```

화면의 안내에 따라 양력/음력, 생년월일, 시각, 성별을 입력하면 됩니다.

### 명령행 모드

```
# 양력
python saju.py --date 1990-03-15 --time 14:30 --gender m

# 음력
python saju.py --date 1988-05-12 --time 06:30 --calendar lunar --gender f

# 음력 윤달
python saju.py --date 2020-04-10 --calendar lunar --leap --gender m

# 진태양시 보정(-30분) 없이 계산
python saju.py --date 1990-03-15 --time 14:30 --gender m --no-correction
```

| 옵션 | 설명 |
|---|---|
| `--date` | 생년월일 (YYYY-MM-DD) |
| `--time` | 출생 시각 (HH:MM, 생략 시 12:00) |
| `--calendar` | `solar`(양력, 기본) / `lunar`(음력) |
| `--leap` | 음력 윤달 여부 |
| `--gender` | `m`(남) / `f`(여) — 대운 순행/역행 계산에 사용 |
| `--no-correction` | 진태양시 보정 생략 |

## 계산 방식

- **음력 변환**: 한국천문연구원 데이터 기반 `korean-lunar-calendar` (1000년~2050년 지원, 윤달 포함)
- **년주**: 입춘(立春) 절입 시각 기준으로 해가 바뀜
- **월주**: 12절기(입춘·경칩·청명…) 절입 시각 기준. 절입 시각은 태양 황경을 천문 근사식(Meeus)으로 직접 계산
- **일주**: 율리우스일 기반, 23시 이후 출생은 다음 날 일주 적용(야자시)
- **시주**: 자시(23:00~00:59)부터 2시간 단위, 일간에 따른 시두법 적용
- **진태양시 보정**: 한국 표준시(동경 135도)와 실제 태양시(서울 기준 약 동경 127도)의 차이 약 30분을 기본으로 반영
- **대운**: 년간 음양 + 성별로 순행/역행 결정, 절입까지의 일수 ÷ 3으로 대운수 산출
- **세운**: 실행 시점의 해(입춘 기준)와 일간의 십신 관계로 올해 운세 풀이

※ 본 프로그램의 풀이는 전통 명리학 이론에 따른 참고용입니다.
